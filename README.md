# DecodeLabs Internship - Task 1: Data Cleaning

## Overview
This project involves cleaning a raw sales dataset (1,200 rows, 14 columns) as part of the DecodeLabs Data Analytics Internship.

## Tasks Performed
- Identified and handled missing values in the `CouponCode` column
- Detected and corrected errors in the `TotalPrice` column
- Validated data types and consistency across all columns
- Exported the cleaned dataset for use in subsequent EDA and SQL analysis

## Tools Used
- Python
- pandas
- Jupyter Notebook

## Files
- `project taskk 1.ipynb` - Data cleaning notebook with code and explanations
- `Cleaned_Dataset.xlsx` - Final cleaned dataset

## Output
A fully cleaned dataset ready for exploratory data analysis (Task 2) and SQL-based analysis (Task 3).
